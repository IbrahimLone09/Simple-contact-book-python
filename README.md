# contact book (python)

A simple contact management system written in python.

## Features

one can use following features using this contact system

1. Add new contacts
2. Search contacts 
3. Display all contacts
4. delete contacts

## Technologies used

1. python
2. Dictionaries
3. loops
4. functions

## How it works

- The program stores contacts using a python
- dictionary where:
- Name = key
- phone number = value

- user can choose options from a menu to manage contacts.

## How to run

1. Download the file
2. open terminal or command prompt
3. run the program
 
- python contact_book.py
